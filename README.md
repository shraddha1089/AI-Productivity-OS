
# 🧠 AI Productivity OS

An AI-powered productivity application built with **Python**, **Streamlit**, and **SQLite**.

The goal of this project is to create a modern productivity platform that combines task management, habit tracking, AI assistance, and productivity analytics into one application while following clean software engineering practices.

---

## 🚀 Current Features

* Multi-page Streamlit application
* Modern landing page
* Task Management module
* Add new tasks
* Priority selection
* Due date selection
* SQLite database integration
* Permanent task storage
* Automatic database initialization
* Responsive two-column layout
